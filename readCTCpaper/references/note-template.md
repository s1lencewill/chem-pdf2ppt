## CTC 领域关键信息提取清单

阅读 CTC 论文时，确保提取并记录以下信息：

### 必提信息
- [ ] 使用的理论方法（泛函/波函数方法层级）
- [ ] 基组及选择理由
- [ ] 色散校正方案（如使用）
- [ ] 溶剂化模型（如涉及溶液相）
- [ ] 计算软件及版本
- [ ] 关键几何参数（键长、键角对比）
- [ ] 关键能量值（反应能、势垒等，含单位）

### 应提信息
- [ ] SCF 和几何优化收敛标准
- [ ] 积分格点设置
- [ ] 频率分析结果（虚频数、ZPE）
- [ ] 过渡态验证（IRC/内禀反应坐标）
- [ ] 多参考诊断（如 T1、<S²>）
- [ ] 与实验或高精度计算的对比验证

### 可选信息
- [ ] 使用的计算资源
- [ ] 输入文件关键内容
- [ ] 辅助分析工具（如 AIM、NBO、EDA 等）
