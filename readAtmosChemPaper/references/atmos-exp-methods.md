# 大气化学实验方法检查清单

## 烟雾箱实验

检查项：

- 箱体材料、体积、温度、相对湿度、压力和混合方式。
- 光源类型、光强、光谱范围和光解频率标定。
- 初始反应物浓度：VOC、NOx、O3、氧化剂前体、种子气溶胶。
- OH/NO3/Cl 等自由基来源和浓度估算方法。
- 反应时长、采样频率、稀释方式和补气方式。
- 壁损校正：气态壁损、粒子壁损、半挥发物壁损。
- 空白实验和对照实验：无光、无氧化剂、无 NOx、无种子粒子等。

重点判断：

- 实验浓度是否高于真实大气很多，是否影响反应通道。
- RH、NOx、种子气溶胶是否覆盖作者声称的环境场景。
- SOA 产率是否做了粒子壁损和气态前体损失校正。

## 材料和附件读取顺序

阅读实验论文时，先自动扫描当前目录和一层子目录，建立材料清单。推荐命名可以加速识别，但不能作为唯一依据。

| 材料 | 常见内容 | 优先用途 |
|---|---|---|
| 主文 PDF | 摘要、主图、核心方法和结论 | 论文主线 |
| Supplementary Information | 实验条件表、补充图、校准、空白、额外产物图 | 证据核查 |
| Source Data | 图表原始数据、Excel、CSV | 数值复核 |
| DOI/期刊页面 | 元信息、图像、SI 下载、数据和代码链接 | 官方来源确认 |

凡是主文把关键证据放在 SI 中，例如 “Supplementary Fig. X shows...”，必须读取 SI 后再写强结论。若 SI 不可用，标注“主文提及，SI 未核查”。

### 自动识别材料角色

| 角色 | 文件名线索 | 内容线索 |
|---|---|---|
| 主文 | article、paper、main、manuscript、正文 | DOI、Abstract、Introduction、References |
| SI | supp、supplementary、SI、ESM、MOESM、supporting、补充 | Supplementary Information、Supplementary Fig/Table/Methods |
| Source Data | source data、sourcedata、figshare、dataset、raw、data | Excel sheet/列名含 Fig、Figure、Table、Supplementary Fig |
| 图片 | figures、images、fig、figure、scheme、mechanism | 图号、图注、机制图、谱图 |

如果文件角色不确定，读取首页或前几页文本后再判断，并在 README 的材料识别表中标注“角色推断”。

## 流动管/反应管实验

检查项：

- 反应管长度、流速、停留时间分布、雷诺数。
- 温度、压力、湿度和壁面材料。
- 反应物注入方式、混合时间和传输损失。
- 氧化剂浓度标定和反应时间定义。
- 是否存在二次反应、壁面反应或扩散限制。

重点判断：

- 停留时间是否足以支持作者给出的动力学解释。
- 速率常数或摄取系数是否区分了气相反应、表面反应和扩散限制。

## 外场观测

检查项：

- 站点位置、季节、气象条件、采样高度和时间分辨率。
- 污染源背景：城市、森林、海洋、农业、燃烧、工业。
- 同步观测物种：VOC、NOx、O3、OH reactivity、PM、气象参数。
- 数据筛选、缺测处理、异常值处理和统计方法。
- 来源解析或模型辅助方法：PMF、CMB、trajectory、box model 等。

重点判断：

- 相关性是否被过度解释为因果。
- 观测时间和地点是否支持作者声称的普遍性。

## 气液界面/异相反应实验

检查项：

- 液相组成、pH、离子强度、有机物浓度和表面活性组分。
- 粒子或液滴尺寸、表面积浓度、相态和黏度。
- 气相反应物浓度、传质条件和湿度。
- 摄取系数、反应概率或表观速率的计算方法。
- 是否区分传质限制、表面反应和体相反应。

重点判断：

- 作者给出的 uptake coefficient 是否可外推到真实大气粒子。
- pH、离子强度和相态是否是机制结论的关键前提。
